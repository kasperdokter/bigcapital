## Deploy on Coolify

1. In Coolify, go to your project and click **New Resource**
2. Choose **Docker Compose (Empty)**
3. Paste the contents of [`bigcapital.yaml`](./bigcapital.yaml), or use the raw URL directly:
   ```
   https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/bigcapital.yaml
   ```
4. Fill in the required environment variables (see below)
5. Deploy

## Required environment variables

These must be set before deployment or the server won't start:

| Variable | Description |
|---|---|
| `MAIL_HOST` | SMTP hostname |
| `MAIL_USERNAME` | SMTP username |
| `MAIL_PASSWORD` | SMTP password |
| `MAIL_FROM_ADDRESS` | Sender email address |

Everything else is either auto-generated by Coolify or has a sensible default.

## Auto-generated by Coolify

Coolify automatically generates and manages these ‚Äî you don't need to set them:

- `SERVICE_USER_MYSQL` / `SERVICE_PASSWORD_MYSQL` ‚Äî database credentials
- `SERVICE_PASSWORD_MYSQLROOT` ‚Äî MariaDB root password
- `SERVICE_PASSWORD_64_JWT` ‚Äî JWT secret
- `SERVICE_URL_WEBAPP` ‚Äî public URL of your Bigcapital instance

## Optional integrations

All disabled by default. Set only what you need:

- **Exchange rates** ‚Äî `EXCHANGE_RATE_SERVICE`, `OPEN_EXCHANGE_RATE_APP_ID`
- **Plaid bank feeds** ‚Äî `BANK_FEED_ENABLED`, `PLAID_ENV`, `PLAID_CLIENT_ID`, `PLAID_SECRET`
- **S3 storage** ‚Äî `S3_REGION`, `S3_ACCESS_KEY_ID`, `S3_SECRET_ACCESS_KEY`, `S3_ENDPOINT`, `S3_BUCKET`
- **Stripe payments** ‚Äî `STRIPE_PAYMENT_SECRET_KEY`, `STRIPE_PAYMENT_PUBLISHABLE_KEY`, etc.
- **Lemon Squeezy** ‚Äî `LEMONSQUEEZY_API_KEY`, `LEMONSQUEEZY_STORE_ID`, etc.

## Signup controls

By default signup is open. To restrict access:

| Variable | Default | Description |
|---|---|---|
| `SIGNUP_DISABLED` | `false` | Set to `true` to disable all signups |
| `SIGNUP_ALLOWED_DOMAINS` | ‚Äî | Comma-separated list of allowed email domains |
| `SIGNUP_ALLOWED_EMAILS` | ‚Äî | Comma-separated list of allowed email addresses |
| `SIGNUP_EMAIL_CONFIRMATION` | `false` | Require email confirmation on signup |
